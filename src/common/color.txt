// 目的：统一管理终端颜色输出
// 使用频率：★★★★☆（大多数有输出的工具都会用到）

// 包含的功能：
// - 颜色定义：RED, GREEN, BLUE, CYAN 等常量
// - 颜色函数：print_red(), print_green(), print_blue()
// - 样式控制：bold(), underline(), blink()
// - 颜色安全检查：支持非TTY终端自动禁用颜色
// - 主题支持：可切换不同颜色主题

// 哪些工具会用到：
// tkls.c    - 文件类型不同颜色
// tkgrep.c  - 搜索关键词高亮
// tkinfo.c  - 美观的系统信息（彩色表格）
// tkmon.c   - 监控指标颜色（CPU红色告警等）
// tkcode.c  - 代码统计颜色区分
