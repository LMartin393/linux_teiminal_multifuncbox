// 目的：长时间操作的进度反馈
// 使用频率：★★☆☆☆（少数工具需要）

// 包含的功能：
// - 进度条创建：create_progress_bar()
// - 进度更新：update_progress(percentage, message)
// - 多类型进度条：条形、旋转、百分比
// - 速度计算：显示剩余时间、传输速度
// - 断点续传支持：保存进度状态

// 哪些工具会用到：
// tkcpmv.c   - 文件复制/移动进度
// tkencode.c - 大文件编码转换进度
// tkconvert.c - 批量格式转换进度
// tkfind.c   - 大目录搜索进度（可选）
