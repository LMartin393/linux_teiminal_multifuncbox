终端多功能工具箱
1. 项目背景
Linux终端是系统管理员的主要工作环境。开发一个集成多种实用功能的终端工具箱，提高工作效率。
2. 项目目标
开发一个类似busybox但更现代化的多功能命令行工具集。
3. 核心功能要求
•	文件操作工具
o	增强版ls（彩色输出、图标、git状态）
o	智能cp/mv（进度条、断点续传）
o	增强版find（正则表达式、内容搜索）
o	文件比较和合并工具
•	文本处理工具
o	增强版grep（正则表达式、上下文、高亮）
o	流式文本处理工具
o	CSV/JSON文件查看和查询
o	编码转换工具
•	系统信息工具
o	美观的系统信息显示
o	实时系统监控（top替代品）
o	网络状态查看工具
o	硬件信息检测工具
•	开发辅助工具
o	代码统计工具
o	文件格式转换工具
o	正则表达式测试工具
o	网络调试工具
4. 技术要求
•	主要语言：C + Python
•	终端支持：ANSI转义序列、Unicode
•	跨平台：主要支持Linux，可扩展macOS
•	安装方式：单个二进制或多模块
